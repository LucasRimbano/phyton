# Práctica de Matemática 3 - UNSAM

Este repositorio reúne mis ejercicios, apuntes y prácticas de **Matemática 3** de la **Universidad Nacional de San Martín (UNSAM)**.

A lo largo de la materia trabajé con distintos temas de matemática y análisis de datos, utilizando principalmente **Python** y **Jupyter Notebook** como herramientas de apoyo para resolver ejercicios, visualizar resultados y reforzar conceptos vistos en clase.

## Contenidos trabajados

Entre los temas que fui practicando en este repositorio se encuentran:

- Álgebra
- Matrices
- Probabilidad
- Teorema de Bayes
- Estadística descriptiva
- Cuartiles y boxplot
- Gráficos y visualización de datos
- Análisis de datasets de práctica

## Objetivo del repositorio

El objetivo de este repositorio es guardar mi progreso, mi práctica personal y los ejercicios que fui resolviendo durante la cursada, tanto para estudiar como para repasar antes de parciales y trabajos prácticos.

También funciona como una forma de documentar cómo fui aplicando herramientas de programación a problemas matemáticos y estadísticos.

## Herramientas utilizadas

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib

## Estructura general

En este repositorio se pueden encontrar:

- Datasets usados para análisis
- Gráficos generados a partir de los ejercicios
